# coc-blade-formatter

blade-formatter (Laravel Blade formatter) extension for coc.nvim

## Install

`:CocInstall coc-blade-formatter`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-blade-formatter-keymap)`

## Lists

`:CocList demo_list`
